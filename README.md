# getcourseAmoBizon(в разработке)

Набор скриптов для интеграции между сервисами Getcourse, AMOcrm, Bizon 365

Логика такая - авторизованный пользователь на Геткурсе совершает переход в вебинарную комнату на Bizon365.
В этот момент в отрабатывает скрипт, который создает сделку+контакт в опрпеделенной воронке в AMOcrm

После окончания вебинара из Bizon365 улетает хук на скрипт, который получает id вебинарной комнаты - по нему достает отчет из базы,
делает выборку нужных пользовательских данных и обрабатывает их по степени "теплоты" (кликал на кнопки на вебинарах, смотрел до конца, писал комментарии и т.д.)

Данная распределенная выборка отправляется в АМОcrm - происходит сопоставление пользователей вошедших на веб, тегирование (горячий, теплый, холодный) и перенос на этап распределения сделок
